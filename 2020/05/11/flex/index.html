flex

flex 笔记

flex 基本概念

  • 开启 flex 布局名称是 flex-container
  • flex-container 直接子元素是 flex-items
  • flex 可以设置为 inline-flex 或 flex

flex 布局模型

  • 主轴(main axis)、交叉轴(cross axis)
    • 主轴大小(main size)、主轴开始(main start)、主轴结束(main end)
    • 交叉轴大小(cross size)、交叉轴开始(cross start)、交叉轴结束(cross end)

flex 相关属性

flex container

  • flex-direction -> 决定主轴的方向

    • row(默认值)
    • row-reverse
    • column
    • column-reverse

  • flex-flow -> flex-direction 和 flex-wrap 缩写

  • flex-wrap (控制 flex 是否在同一行显示)

    • wrap
    • nowrap (默认是在同一行显示)
    • wrap-reverse

  • justify-content -> 决定了 flex items 在主轴上的对齐方式

    • flex-start(默认值)
    • flex-end
    • space-around
    • space-between
    • space-evenly

  • align-items -> 决定了 flex items 在交叉轴的对齐方式

    • flex-start
    • flex-end
    • center
    • baseline (以文字基线对齐)
    • normal
    • stretch 当 flex items 在 cross axis 方向的 size 为 auto 时,会自动拉伸至填充 flex container

  • algin-content -> 决定了多行 flex items 在交叉轴上的对齐方式

    • stretch
    • flex-start
    • flex-end
    • center
    • space-between
    • space-around
    • space-evenly

flex items

  • order -> 决定 flex items 显示顺序 值越小越在前面

  • align-self -> 决定 flex items 在交叉轴的对齐方式

  • flex-grow

  • flex-shrink

  • flex-basis 决定 flex items 在主轴上的 base size

  • flex

    • 1 个参数 类似于 flex grow
    • 2 个参数
      • 第一个参数 flex-grow
      • 第二个参数 如果有单位则是 flex basis 否则是 flex shrink
    • 3 个参数
      • flex-grow flex-shrink flex-basis

移动端一些问题

  • meta=”viewport”

  • 不同单位

    • px 像素
    • em 相对于父元素字体大小
    • % 不同场景相对于的不一样
    • rem 相对于 root(html)元素的字体大小

  • rem 适配

一些 problem

  • margin-top: 50% 居然相对于父元素的宽度
  • 媒体查询

@meida srceen and (min-width:357px) {
html {

}
}
